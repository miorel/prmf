This method will return 1 or 0 based on whether it determines 2 colors to be near enough to each other There can be a set tolerance passed to this function which will act as factor to determine the result. If a tolerance is used then the result is based on the color distance between the given, which is simply calculated as the distance between the points (since color can be interpreted as such).
